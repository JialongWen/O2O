# 校园商铺平台
主要功能设计与总结
============
项目设计
---------
## 功能设计
### 前端展示系统
* 1.头条展示
* 2.店铺类别展示
* 3.区域展示
* 4.店铺<br>
    a.列表展示
    b.查询
    c.详情
* 5.商品<br>
    a.列表展示
    b.查询
    c.详情
* 6.个人信息(此处为后扩展功能，可能没有源码）<br>
    a.消费记录
    b.各店铺积分查询
    c.兑换记录
    <br>
### 店家系统
* 1.local账号维护
* 2.微信账号维护
* 3.店铺信息维护
* 4.权限验证
* 5.商铺类别维护<br>
